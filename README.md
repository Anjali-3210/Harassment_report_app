🛡️ Harassment Report App

A secure and user-friendly platform that enables individuals to report harassment incidents safely and anonymously. The application helps organizations, colleges, and workplaces manage complaints efficiently while ensuring privacy, transparency, and accountability.

📌 Overview

The Harassment Report App is designed to provide a safe digital space for victims or witnesses to report harassment incidents without fear.
It simplifies the complaint process by allowing users to submit reports, attach evidence, and track their case status, while administrators can manage and resolve cases effectively.
The goal of the system is to encourage reporting, improve response time, and promote safer environments.

✨ Features

👤 User Features

🔹 Submit Harassment Reports
Users can easily report incidents through a structured complaint form.

🔹 Anonymous Reporting
Users can submit reports without revealing their identity to maintain privacy.

🔹 Upload Evidence
Attach screenshots, images, or documents to support the complaint.

🔹 Track Complaint Status
Users can monitor the progress of their submitted report.

🛠️ Admin Features

🔹 View All Reports
Administrators can access and review all submitted harassment complaints.

🔹 Case Management System
Admins can update report status such as:

⏳ Pending

🔍 Under Investigation

✅ Resolved

🔹 Evidence Review
Admins can examine uploaded files and supporting documents.

🔹 User Management
Manage users and their submitted reports.
